## Ingredient Information

### Active Ingredients by Product

**Vitamin C Products:**
- Vitamin C Brightening Serum: 15% L-Ascorbic Acid, Vitamin E, Ferulic Acid
- Vitamin C Mask: 10% Ascorbyl Glucoside, Niacinamide
- **Note:** Both contain Vitamin C but different forms with different stability profiles

**Retinol Products:**
- Retinol Night Serum: 0.5% Retinol, Squalane, Ceramides
- Night Cream: 0.1% Retinol (lower concentration for daily use)
- **Important:** See usage warnings below

**Exfoliating Acids:**
- AHA/BHA Exfoliating Toner: 7% Glycolic Acid, 2% Salicylic Acid, Niacinamide
- Exfoliating Gel Cleanser: 2% Salicylic Acid
- **Note:** Different concentrations for different usage frequencies

**Niacinamide:**
- Niacinamide Serum: 10% Niacinamide, Zinc PCA
- AHA/BHA Exfoliating Toner: 2% Niacinamide (supporting ingredient)
- Vitamin C Mask: 4% Niacinamide
- **Cross-reference:** Compatible with most actives (see compatibility section)

**Hyaluronic Acid:**
- Hyaluronic Acid Serum: Multi-weight HA complex
- Daily Hydrating Moisturizer: Contains HA (not listed as primary active)
- Hydrating Cream Cleanser: Low-weight HA

**Peptides:**
- Peptide Firming Serum: Matrixyl 3000, Argireline, Copper Peptides
- Eye Cream: Palmitoyl Tripeptide-1
- Night Cream: Palmitoyl Pentapeptide-4

### Fragrance Information

## Product Compatibility

### Can Use Together

**Niacinamide + Retinol:**
- Yes - recommended pairing
- Use Niacinamide Serum in AM, Retinol Night Serum in PM
- OR layer Niacinamide first, then Retinol in PM routine
- Niacinamide helps reduce retinol irritation

**Vitamin C + Hyaluronic Acid:**
- Yes - excellent combination
- Apply Vitamin C Brightening Serum first, then Hyaluronic Acid Serum
- HA helps with Vitamin C absorption

**Vitamin C + Niacinamide:**
- Yes - despite old information suggesting conflict
- Modern formulations are pH-stable
- Can layer or use at different times of day

**Retinol + Peptides:**
- Yes - use in same PM routine
- Peptides first, then retinol
- Both work on anti-aging pathways without conflict

### Use With Caution

**AHA/BHA + Retinol:**
- Can use but not on same night initially
- Alternate nights: AHA/BHA one night, Retinol next night
- Advanced users can layer after 4-6 weeks of tolerance
- **High sensitivity risk** - start slow

**AHA/BHA + Vitamin C:**
- pH conflict possible
- Use Vitamin C in AM, AHA/BHA in PM
- Do not layer in same routine

**Multiple Exfoliants:**
- AHA/BHA Toner + Exfoliating Gel Cleanser: Use toner only 2-3x/week if using cleanser daily
- Over-exfoliation risk: Redness, sensitivity, compromised barrier
- Recommend: Pick one exfoliant product or alternate days

### Do Not Combine

**Retinol + Exfoliating Gel Cleanser (Salicylic Acid):**
- High irritation risk when used together
- OK to use cleanser in AM, retinol PM
- NOT OK to use cleanser then retinol in same PM routine
- **Exception:** Advanced users after 8+ weeks of retinol tolerance

## Product-Specific Questions

### "Which cleanser should I recommend?"
- **Oily/acne-prone:** Exfoliating Gel Cleanser (has salicylic acid)
- **Dry/sensitive:** Hydrating Cream Cleanser
- **Normal/combo:** Gentle Foaming Cleanser
- **Makeup removal:** Oil Cleanser (double cleanse method)

### "Which moisturizer for oily skin?"
- **Best:** Oil-Free Gel Moisturizer (lightweight, non-comedogenic)
- **Alternative:** Daily Hydrating Moisturizer (lighter than barrier cream)
- **Avoid:** Rich Barrier Cream, Night Cream (too heavy)

### "Is the Mineral Sunscreen greasy?"
- Yes, slight white cast and thicker texture
- Best for: Sensitive skin, acne-prone (non-comedogenic)
- **Alternative for oily skin:** Tinted Sunscreen SPF 45 (cosmetically elegant, no white cast)

### "Can pregnant customers use Retinol?"
- **No** - retinol is not pregnancy-safe
- **Alternatives:**
  - Vitamin C Brightening Serum (pregnancy-safe)
  - Peptide Firming Serum (pregnancy-safe)
  - Niacinamide Serum (pregnancy-safe)
- See product FAQ on website for full pregnancy-safe routine

### "What's in the Starter Skincare Set?"
- Contents: Gentle Foaming Cleanser (50ml), Hyaluronic Acid Serum (15ml), Daily Hydrating Moisturizer (30ml)
- **Value:** $89 for $110 worth of products
- **Good for:** New customers, gift purchases, building basic routine
- **Note:** Does not include sunscreen - recommend adding separately

### "Customer says Vitamin C Serum is turning orange/brown"
- **Normal:** Oxidation due to air/light exposure
- **If slight color change:** Still effective but use within 3 months
- **If dark brown:** Offer replacement - product has oxidized
- **Storage tip:** Keep in cool, dark place; refrigeration extends shelf life
- **Check purchase date:** Vitamin C products have 6-month shelf life after opening

## Ingredient Sensitivities & Reactions

### Common Sensitivities

**Essential oils:**
- Found in: Oil Cleanser, Overnight Sleeping Mask
- Symptoms: Redness, itching, rash
- **Action:** Discontinue immediately, offer fragrance-free alternatives
- See Return Policy for adverse reaction handling

**Acids (AHA/BHA):**
- Expected: Slight tingling, temporary redness (normal)
- Concerning: Burning, persistent redness, peeling (over-exfoliation)
- **Action:** Reduce frequency to 1-2x/week or discontinue
- Note: More common in sensitive skin or when combined with retinol

**Retinol:**
- Expected: Initial dryness, flaking (retinization period - normal)
- Concerning: Severe irritation, burning (concentration too high)
- **Action:** Switch to Night Cream (lower retinol %) or stop use
- Recommend: Start 2x/week, increase gradually

**Niacinamide:**
- Rare sensitivity (niacinamide is generally well-tolerated)
- Some customers report flushing at 10% concentration
- **Action:** Recommend products with lower niacinamide % (AHA/BHA Toner has 2%)

### If Customer Reports Adverse Reaction

1. **Immediate:** Stop using product
2. **Document:** Product name, batch number (on bottom of bottle), reaction type
3. **Process:** Full refund immediately (see Return Policy - Adverse Reactions section)
4. **Flag:** Add to product quality review list
5. **Follow-up:** Customer success team will reach out within 48 hours
